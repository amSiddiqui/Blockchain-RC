# Simple Blockchain in Rust

## Overview

This project is a simple blockchain implementation written in Rust. It serves as an educational tool for understanding the basics of blockchain technology and the Rust programming language.

## Features

- Basic Proof-of-Work (PoW) algorithm
- Simple transaction model
- Block creation and validation
- Chain integrity checks

## Prerequisites

- [Rust](https://www.rust-lang.org/tools/install)
- [Cargo](https://doc.rust-lang.org/cargo/getting-started/installation.html) (usually comes with Rust)

## Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/amSiddiqui/Blockchain-RC.git
    ```

2. Navigate to the project directory:

    ```bash
    cd Blockchain-RC
    ```

3. Build the project:

    ```bash
    cargo build
    ```

## Usage

Run the application:

```bash
cargo run
